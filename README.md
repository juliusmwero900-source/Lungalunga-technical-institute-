Lungalunga Online Learning System is a web-based platform designed to support students and staff of Lungalunga Technical Institute. The system provides easy access to learning materials, student registration, results, and academic resources in a simple and user-friendly interface.

🎯 Objectives

To provide students with access to notes and learning materials online

To simplify student registration and login processes

To allow students to view their academic results (CATs, Exams, Assignments)

To improve digital learning within the institution

🚀 Features

🔐 User Authentication (Login & Registration)

📚 Notes Section (Download and view study materials)

📝 Results Section (CATs, Exams, Assignments)

👨‍🎓 Student Dashboard

🌐 Responsive Design (Accessible on phone & computer)
